打通这个闲鱼 pay 算是一环

今天想到一个 idea 是搭建一套 agent，自动从电商平台抓取各硬件（CPU、显卡等） 的价格，汇总推送到社交媒体，现在 agent 这么发达应该不难做，而且肯定是有这个需求的，上大学的时候我经常看 **ZoveyChen** 的硬件底价，挺多人看这个的，不过后来他不更新了

他当时也尝试用了一些自动化，不过 23 年还比较原始 [【硬件底价系列,并不是全网最低价】](https://www.bilibili.com/video/BV1cp4y1P7Db?vd_source=5ca30838bf3c8b357bddca1ff2565902)

然后我就开始折腾这一套怎么搭建，去淘宝京东上搜索关键词（比方说 12400f 、5060ti 这种），基于浏览器的自动化例如 playwright（说到这个，上面闲鱼的也是用这个）（当然除了 playwright 还有很多其他方法，最近看到很多这方面的项目，竞争挺激烈的），是可以做到的，但是很多低价是在 pdd 的，pdd 众所周知是非常封闭，web 端是没有的，然后我就想让 agent 操作安卓模拟器或者安卓真机，这方面可能还不是很成熟？adb 是能拉到 UI tree 的比较方便，但是其实很多国内 app 已经在避免这个了，GitHub 也没有很多逆向 pdd 的项目

并且这个价格很多时候是和打折券、国补挂钩的，这也是一个问题

然后我去看 B 站有没有人做这个 niche，发现已经有了，比较好的是 芝士华莱士 的显卡日报，并且不止价格，后面还会跟一段新闻，观感很好，现在有 30w 粉（我之前听说一个经验说法是 UP 主的年收入大概就是粉丝量，当然这个也会根据不同分区不一样）（开个题外话，我之前在 V 站刷到帖子，那个人是 B 站的星团菌，靠 AI 做哲学或者思辨类视频，也能做到几十万粉）

当然其他垂类可能也有机会，但是我还没有观察到，目前先把这个点子搁浅在这里

还有个点子是想做自动关注电商平台购物车里的商品降价通知，毕竟我想买自行车，之前想的是迪卡侬的 RC100（其他的喜德盛 RC200 其实也可以），但是现在 1.5k 的价格不是很友好，之前有看见更便宜的

但是这个功能其实京东也有，但是藏的比较深在二级菜单里面（也不知道他们的产品经理会不会关注这个 feat 的使用率），之前一直不知道，感觉简陋感觉可能够用？不过不知道这方面能不能做一些花活。京东淘宝都购物车或者说商品收藏是可以导出的，这样我就可以拿到 sku 的 list（京东其实就是一个短链，跳转时候直接把 sku 拼到 query 里面），然后可以追踪

确实不知道有没有什么活可以整，比方说能看到这个商品自动看全网最低价有没有降（不过这个全网最低价也有很多成熟平台了，高中时候就有）
